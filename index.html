Python Genetic Algorithm

Fully scratch-built in Python, this algorithm simulates biological evolution to determine the optimal solution to a given challenge, such as the Knapsack Problem. The Knapsack Problem is a simple puzzle in which a number of items, each with a certain weight and value, must be combined to produce the highest possible value without exceeding a weight limit. The algorithm solves this by the following procedure:
1) Create a genome (represented by an array of boolean values). Similar to an individual organism in nature.
2) Create a population of genomes (the original generation).
3) Determine the fitness of each genome for a problem.
4) Genomes are pitted together and the winners (determined by their fitness score) are saved.
5) Chromosomes in a genome undergo crossover operation and mutation.
6) Generate new population based on the results of the mutation and crossover.
7) Repeat for the specified number of generations.
